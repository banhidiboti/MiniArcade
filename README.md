# Mini Arcade (Workspace)

Ez a workspace a teljes Mini Arcade projektet tartalmazza.

## Hol van a fő app?

A tényleges Vite + Vue alkalmazás ebben a mappában van:

- `mini-arcade/`

Részletes dokumentáció, játékleírások, irányítás és futtatási lépések:

- `mini-arcade/README.md`

## Gyors indítás

```bash
cd mini-arcade
npm install
npx vite
```

Build:

```bash
npm run build
```

Deploy (GitHub Pages):

```bash
npm run deploy
```
