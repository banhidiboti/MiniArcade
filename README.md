# Mini Arcade Workspace

This workspace contains a Vue 3 + Vite arcade game project.

## Where the app lives
The playable app is in:

- `mini-arcade/`

Run all app commands from that folder.

## Quick Start

```bash
cd mini-arcade
npm install
npx vite
```

## Build & Deploy

```bash
cd mini-arcade
npm run build
npm run deploy
```

## Included Games
- Brick Breaker (Classic + Endless)
- Cyber Snake

For full gameplay details, controls, and project structure, see:

- `mini-arcade/README.md`
